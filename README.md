# type-single-page
 This Project was Made by Grace Kelley about David Jonathan Ross's font 'Megascope V2'. it is a variable font available on his Website.
## process
I created a mockup in figma and used a class starter code that I modified with my own grid and elements.
## Attribution
The font I used was Megascope V2, made by David Jonathan Ross.
## Accessability
This site uses large, clear letterforms to increase readability as well as short, consice text paired with images with alt text for screen readers.
